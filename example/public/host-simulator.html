Mock host: extension refund

This page imitates Render (parent): it loads the example app in an iframe and sends extensionRefundRequest — the same message the real POS sends when refunding a redeem payment. The example app registers installExtensionRefundListener in main.tsx with a mock provider refund.

Run npm run dev in example/, then open this file at http://localhost:5179/host-simulator.html (must match the iframe origin).
